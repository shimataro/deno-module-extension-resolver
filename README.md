# deno-module-extension-resolver

Resolves module extensions in `import` / `export from`.

This is a **NPM package** for deno.

## Install

```bash
npm i deno-module-extension-resolver
```

## Usage

```bash
deno-module-extension-resolver SRC_DIR DST_DIR
```
